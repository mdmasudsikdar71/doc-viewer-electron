## Doc Viewer

A simple **Electron** application that uses **Mozilla PDF.js** to view local PDF files offline.

---

### Install

```bash
npm install
npm install pdfjs-dist
npm start
```

---

### Usage

1. Run the app.
2. Click **“Select PDF to view”**.
3. Choose any `.pdf` file to open it in the viewer.

---

### Tech Stack

* Electron
* PDF.js (pdfjs-dist)
